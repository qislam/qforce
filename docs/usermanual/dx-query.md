Run a SOQL and save results to csv.

### Usage

```bash
  $ qforce dx:query
```

### Options

```bash
  -f, --file=file          Relative path of query file in unix format.
  -h, --help               show CLI help
  -q, --query=query        SOQL query as string.
  -r, --result=result      Relative path to save results of query.
  -u, --username=username
```
