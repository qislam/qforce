Execute anonymous apex.

### Usage

```bash
  $ qforce dx:exe
```

### Options

```bash
  -f, --file=file          Relative path of apex file in unix format.
  -h, --help               show CLI help
  -r, --result=result      Relative path to save results.
  -u, --username=username
```

### Aliases

  $ qforce exe
  $ qforce dx:exe
