Open an org.

```
USAGE
  $ qforce dx:open

OPTIONS
  -h, --help               show CLI help
  -p, --path=path
  -u, --username=username

ALIASES
  $ qforce open
  $ qforce dx:open
  $ qforce o

EXAMPLE
  $ q dx:open -u uat
```
