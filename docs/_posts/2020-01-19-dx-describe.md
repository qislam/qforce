A wrapper for sfdx command to get description of an object.

```
USAGE
  $ qforce dx:describe

OPTIONS
  -a, --all                To get all sObjects.
  -h, --help               show CLI help
  -r, --result=result      Relative path to save results.
  -s, --sobject=sobject    sObject name.
  -u, --username=username

ALIASES
  $ qforce describe
  $ qforce dx:describe
```
